boxWidth:
  Table 容器的宽度，当Table的宽度大于Table容器，出现内滚动条，滚动条移动时，Head、Food保持固定。

boxHeight:
  Table 容器的高度，当Table的高度大于Table容器，出现内滚动条，滚动条移动时，Head、Food保持固定。

width:
  Table 宽度

height:
  Table 高度

fixedColumns:
  固定列的Index，也可以在需要固定列的，Th上加Fixed标示，在此列前（包含次列）将会被固定。

cloneHeadToFoot:
  复制Head到Foot，如果Table中没有Foot，设置此属性为true，将会复制Head到Foot，并且对于不对称的Head，将会自动翻转。

autoScroll:
  为True时，只有鼠标经过时才会显示滚动条，若果为false，滚动条将会一直显示。
