使用 JgWidgets 一般推荐不再单独写脚本,而是通过插件注册的方式进行开发
如果实际业务当中需要使用自定义的方法,JgWidgets Page , Window ，Tabs也提供一系列的回调方式
使用起来类似 jQuery 的
$(function(){
});
异步执行加载完成后会自动执行,并且会传入加载页面的 Html 作用域对象,方便获取
当前加载的元素

$.jgPage("onOpen",function($content){
alert($content.html());
});

$.jgWindow("onOpen",function($content){
alert($content.html());
});

$.jgTabs("onOpen",function($content){
alert($content.html());
});

也可以使用通用的方法，同时支持Window ，Page ，Tabs

$.jgReady(function($content){
//页面异步加载完成后将执行
alert($content.html());
});
